# Changelog

## 3.0.0

- Полностью переработан пользовательский сценарий.
- В обычном режиме больше не спрашивается порт российского сервера.
- Пользователь вставляет один зарубежный Endpoint, а входящий порт автоматически совпадает с целевым.
- Интерфейсы скрыты из обычного мастера и определяются автоматически.
- Добавлен парсер строк `Endpoint = IP:PORT`.
- Добавлена готовая строка Endpoint для клиентского конфига.
- Добавлена поддержка hostname с повторным DNS-разрешением.
- Добавлены транзакционные изменения и автоматический откат.
- Добавлены резервные копии конфигурации.
- Добавлены диагностика маршрута, hooks, counters и конфликтов локальных портов.
- Добавлены зеркальный и advanced-режимы.
- Улучшены установка, обновление и CLI.

## 2.1.0

- Исправлен пропуск вопросов о портах.
- Автоматизирован выбор интерфейсов.
